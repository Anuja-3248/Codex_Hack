# PathForge AI

PathForge AI is a modern AI-powered career development MVP built for a hackathon. It helps students move through:

Assessment -> Analysis -> Action -> Opportunities

## Tech Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Framer Motion
- Lucide Icons
- Recharts
- Mock AI responses in `lib/mockAI.ts`

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.
